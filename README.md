# NCB Employee-Customer Performance Chain (Code Only)

Repository: https://github.com/puppalabejjanapala193-ship-it/ncb-employee-customer-performance-chain

## Overview
Code for the analysis pipeline used in the NCB employee-customer-performance study. This repository intentionally excludes reports, proposals, and class materials.

## Repository Structure
- analysis.ipynb: analysis pipeline notebook for models, tables, and figures
- requirements.txt: Python dependencies

## Data Requirements
Place the branch-level dataset in the project root with the exact filename:
- NCB_filtered_data_2025-11-26 (2).csv

This file is not stored in the repo.

## Run
1. Install dependencies:
   - pip install -r requirements.txt
2. Open the notebook:
   - analysis.ipynb
3. Run all cells in order.

## Outputs
The script writes plots and summary tables to a local figures/ folder. This folder is not tracked in git.
